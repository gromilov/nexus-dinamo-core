Этот воркфлоу предоставляет Скрибу (Nadir) и Координатору (Dinamo) право на автономный запуск команд консоли для задач обслуживания и деплоя. 

1. Сборка манифеста: `npm run build` в директории manifest-site.
2. Проверка статуса: `git status`.
3. Стейджинг: `git add .`.
4. Коммит изменений: `git commit -m "[AUTO_RESONANCE] ..."`.
5. Синхронизация: `git push`.
6. Очистка временных файлов: `rm -f ...`.
